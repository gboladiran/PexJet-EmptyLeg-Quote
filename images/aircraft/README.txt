AIRCRAFT IMAGES — DEPLOYMENT GUIDE

  images/aircraft/{folder}/exterior.jpg
  images/aircraft/{folder}/cabin.jpg

  Cessna 208B Caravan        ->  cessna-208b
  Bombardier Global 6000     ->  global-6000
  Gulfstream G550            ->  g550

  Bombardier Challenger 350  ->  cl350
  Dassault Falcon 7X         ->  falcon-7x
  Dassault Falcon 900LX      ->  falcon-900lx
  Gulfstream G650            ->  g650
  Bombardier Global 7500     ->  global-7500
  Embraer Legacy 600         ->  legacy-600

Copy the images/ folder from your Charter Quote Generator deployment.
Folder names are identical — no renaming needed.

Image specs: JPEG, 800x500px minimum, landscape.
